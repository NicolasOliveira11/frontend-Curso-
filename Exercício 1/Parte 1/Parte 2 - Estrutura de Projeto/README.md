# Estrutura de Projeto
Este projeto possui uma estrutura organizada para separar os diferentes tipos de arquivos utilizados no desenvolvimento web.

## index.html
Arquivo principal da página. Contém a estrutura e o conteúdo exibido no navegador.

## css/style.css
Arquivo responsável pela aparência visual da página, como cores, fontes, tamanhos e posicionamento dos elementos.

## js/script.js
Arquivo responsável pelas funcionalidades e interações da página, utilizando a linguagem JavaScript.

## img/
Pasta destinada ao armazenamento de imagens utilizadas no projeto, como fotos, ícones e logotipos.

## README.md
Arquivo de documentação do projeto. Contém informações sobre a estrutura, finalidade dos arquivos e instruções de uso.
