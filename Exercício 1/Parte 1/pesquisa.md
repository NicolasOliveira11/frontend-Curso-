# Exercício 1 – Parte 1 - Pesquisa

## O que é Frontend?
Frontend é a parte de um site ou aplicação que o usuário vê e com a qual interage. Ele inclui elementos como botões, menus, imagens, textos e formulários. O objetivo do frontend é proporcionar uma boa experiência para o usuário.

## O que é Cliente x Servidor?
O modelo Cliente-Servidor é uma forma de comunicação entre computadores.
* **Cliente:** é o dispositivo ou programa utilizado pelo usuário, como um navegador (Google Chrome, Firefox, Edge).
* **Servidor:** é o computador que armazena os dados e responde às solicitações feitas pelo cliente.
Quando um usuário acessa um site, o navegador envia uma requisição ao servidor, que devolve as informações necessárias para exibir a página.

## Qual é o papel do HTML, CSS e JavaScript?
### HTML
HTML (HyperText Markup Language) é a linguagem responsável pela estrutura da página. Ele define títulos, parágrafos, imagens, tabelas e outros elementos.
### CSS
CSS (Cascading Style Sheets) é utilizado para estilizar a página, definindo cores, tamanhos, fontes, espaçamentos e posicionamento dos elementos.
### JavaScript
JavaScript é a linguagem responsável pela interatividade da página. Com ela é possível criar animações, validar formulários, atualizar conteúdos dinamicamente e muito mais.

## O que é W3C?
W3C (World Wide Web Consortium) é uma organização internacional responsável por desenvolver padrões para a Web. Seu objetivo é garantir que sites e aplicações funcionem corretamente em diferentes navegadores e dispositivos.
