# Projeto Frontend – Sousou no Frieren (Parte 5)

## Objetivo
Este projeto tem como objetivo praticar a criação de páginas web utilizando HTML semântico, incluindo estruturação de conteúdo, listas, links, imagens e interatividade básica com JavaScript.

## Estrutura de pastas

```
projeto/
│
├── index.html
└── img/
    └── (imagens do projeto, se houver)
```
## Como abrir o projeto

1. Baixe ou clone o repositório.
2. Abra a pasta do projeto no VS Code.
3. Clique duas vezes no arquivo `index.html` ou utilize a extensão Live Server.
4. A página será aberta no navegador.

## Validação e revisão

Durante o desenvolvimento foram aplicadas as seguintes práticas:

* Uso de HTML semântico (`header`, `nav`, `main`, `section`, `footer`)
* Estruturação correta de listas (`ul` e `ol`)
* Uso adequado de links externos e internos
* Inserção de imagens com atributo `alt`
* Implementação de interação básica com JavaScript
* Organização e revisão do código para melhor legibilidade

O código foi revisado para evitar erros de estrutura e garantir boas práticas de desenvolvimento web.
