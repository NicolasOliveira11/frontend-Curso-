## Objetivo

Adaptar uma página desenvolvida com HTML semântico para utilizar o modelo **table-based layout**, técnica utilizada antigamente para organizar páginas da web antes da popularização do CSS.

## Por que essa técnica é antiga?

O layout baseado em tabelas era muito utilizado no início da web porque o CSS ainda possuía poucos recursos para posicionamento de elementos. Atualmente essa técnica é considerada ultrapassada, pois as tabelas foram criadas para exibir dados tabulares, e não para estruturar páginas.

Hoje, o recomendado é utilizar elementos semânticos do HTML juntamente com CSS, utilizando recursos como Flexbox e Grid Layout.

## Conclusão

Embora o layout baseado em tabelas funcione, ele não é indicado para projetos atuais devido à baixa flexibilidade, manutenção mais difícil e pior acessibilidade quando comparado às práticas modernas de desenvolvimento web.
